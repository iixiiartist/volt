Why Volt?

Built for production, designed for developers. Volt redefines what's possible with AI agents.

🔄

Dynamic RAG Loop

Tools, skills, and memories are retrieved via pgvector cosine similarity — not hardcoded. Up to 75% fewer input tokens per LLM call.

📦

Compiled Manifest Pattern

Author skills in Markdown (SKILL.md), compile into PostgreSQL with HNSW indexing. Human-friendly authoring, machine-optimized runtime.

🤖

Multi-Agent Orchestration

Parallel, pipeline, and supervisor patterns built-in. Coordinate multiple agents to solve complex problems together.

🔒

Permission System

Destructive operations require human approval. Sandboxed execution for provisioned tools with full audit logging.

⚡

Zero Dependencies

Single 18MB Rust binary. No Python, Node, or Docker required at runtime. Fast cold start <100ms.

🔌

MCP Native

Full Model Context Protocol support for tool interoperability. Connect to any LLM provider seamlessly.
