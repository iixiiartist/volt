Core Specifications

Built for Production Systems

🔄

Unified RAG Loop

Tools, skills, and long-term memory are dynamically searched via pgvector cosine similarity. Filters out execution noise and drops input costs by up to 75%.

📦

Compiled Manifest Pattern

Author agent instructions in human-friendly Markdown (SKILL.md). Volt compiles assets directly into structured PostgreSQL relational datasets.

🔒

Kernel-Level Isolation

Polyglot tools (Python, TS, Mojo, Bash) are executed within native Linux unshare namespaces and bound by strict prlimit guardrails.

🛡️

Pre-Execution Validation

Volt intercepts every tool call, parsing arguments against a structured local denylist to neutralize token injection attacks before thread assignment.

⚡

Statically Linked Binary

A single ~18MB executable with zero dependencies. Avoids the interpreter tax, version conflicts, or heavy node_modules layouts.

🔌

Model Context Protocol

Implements native Model Context Protocol (MCP) clients and servers directly over Tokio async channels, removing execution blocking hooks.
